The SWIFT Project

The Global Backbone of Financial Communication

Project Overview

The SWIFT Project provides a secure network that allows more than 11,000 financial institutions in over 200 countries to send and receive information about financial transactions in a secure, standardized, and reliable environment.

Key Initiatives

SWIFT gpi

Enhancing cross-border payments with real-time tracking, transparency, and lightning-fast speed.

ISO 20022

Migrating to a richer data standard to improve automation and compliance across the global economy.

Security and Innovation

Security is the top priority of the SWIFT project. Through the Customer Security Programme (CSP), SWIFT helps the community strengthen its cyber defenses and share critical threat intelligence.
